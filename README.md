一个TacZ的附属模组，实现以下功能：

--枪械在命中实体腿部时只造成60%伤害

--可配置的全自动模式下首发子弹后坐力倍率

--可配置的连射惩罚机制

--让枪械安装某些配件时无法使用连发模式开火（主要是为了防止连发模式规避配件降低射速的负面效果）

--修复使用跑酷模组的爬行时，tacz的两脚架不会放下的bug

--修复tacz addon的解放配件对arcana的“模块与皮肤”无效的bug

--修复tacz arcana的镜内放大功能导致机瞄和红点瞄具无效的bug

--修复同时安装parcool和tacz模组后，在疾跑时开镜导致抽搐的bug

--为gd656peek模组添加探头自动开镜功能

所有功能均可配置

必要的前置：Timeless and Classics Zero 1.1.6或更高版本
