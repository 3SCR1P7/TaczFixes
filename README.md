这是一个TaCZ的附属模组。

实现了以下功能：

--首发后坐力倍率机制

--四肢伤害衰减机制

--连射惩罚机制

--开火击退机制

--枪械升级机制

--跳弹机制

为TaCZ及其附属添加了以下选项：

--子弹产生的爆炸仅击退子弹

--枪械安装指定配件后禁用连发模式

--探头自动开镜 (需要gd656peek)

为TaCZ及其附属修复了以下bug：

--使用跑酷模组的爬行时，tacz的两脚架不会放下

--tacz addon的解放配件对arcana的“模块与皮肤”无效

--tacz arcana的镜内放大功能导致机瞄和红点瞄具无效

--同时安装parcool和tacz模组后，在疾跑时开镜导致抽搐

一切功能均可配置。

必要的前置：Timeless and Classics Zero 1.1.6或更高版本。
